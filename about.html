About StockSim

Virtual Stock Market Simulator

StockSim is a fully interactive, frontend-only stock market simulator designed for educational purposes. It provides a risk-free environment to practice trading strategies, understand market dynamics, and learn about portfolio management.

Features

  • Real-Time Price Simulation: Stock prices update every 3 seconds with realistic market movements
  • Buy & Sell Trading: Execute trades with validation and real-time portfolio updates
  • Portfolio Tracking: Monitor your investments with detailed analytics and profit/loss calculations
  • Trade History: Complete transaction log with timestamps and profit/loss tracking
  • Watchlist: Star your favorite stocks for quick access
  • Data Persistence: All data stored locally using LocalStorage

Technologies Used

HTML5
CSS3
Bootstrap 5
JavaScript (ES6+)
LocalStorage API
Canvas API
